# Vikash Kumar — Full Stack & GenAI Engineer Portfolio

![Portfolio Preview](https://img.shields.io/badge/Status-Live-emerald?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Tech-Next.js%20%7C%20TS%20%7C%20Three.js-blue?style=for-the-badge)

A high-fidelity, cinematic developer portfolio designed with an "Engineering Void" aesthetic. Built with **Next.js 14**, **TypeScript**, and **Framer Motion**, featuring real-time 3D elements and professional-grade micro-interactions.

## 🚀 Live Demo
[Check it out here](https://portfolio-vikash.vercel.app) *(Update with your actual link if different)*

## ✨ Key Features

- **Cinematic Hero Section**: Features a metallic 3D orb built with R3F (React Three Fiber) and a custom typewriter state machine.
- **Bento Grid Architecture**: Clean, modular layout for projects and certifications inspired by Linear and Stripe.
- **Premium Animations**: Staggered scroll reveals, magnetic buttons, and glassmorphic hover effects using Framer Motion.
- **Validated Expertise**: Dedicated certifications section with pulsing "Verified" badges and Credly integration.
- **Dark-First Design**: obsidian black backgrounds with tailored violet and cyan gradients for an elite SaaS feel.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) / [Three.js](https://threejs.org/)
- **Icons**: [Lucide React](https://lucide.dev/) / [React Icons](https://react-icons.github.io/react-icons/)

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vikashkumar302004/Portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
